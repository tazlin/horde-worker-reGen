# Advanced setup

The advanced setup guides have moved into the documentation, reorganised by task:

- **[Install from scratch and choose a PyTorch build](docs/how-to/choose-a-pytorch-build.md)**: manual
  virtualenv install, `uv`/`pip` usage, and forcing a specific CUDA/CPU/ROCm build.
- **[Run on AMD ROCm](docs/how-to/run-on-amd-rocm.md)**: Linux ROCm, the DirectML status, and the
  experimental WSL build.
- **[Run headless](docs/how-to/run-headless.md)**: unattended worker and environment-variable config.
- **[Run in Docker](docs/how-to/run-in-docker.md)**: container images.

Start at the [documentation home](docs/index.md), or come to the
[#local-workers](https://discord.com/channels/781145214752129095/1076124012305993768) channel in the
[official Discord](https://discord.gg/3DxrhksKzn) for the latest information or questions.
